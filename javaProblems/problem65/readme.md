# [Problem 65 - Find Greatest Common Divisor of Array](https://github.com/SST-OSS/club-onboarding/issues/65)
The code finds the greatest common divisor of the smallest and the largest element of the array.

```
# Example Input 1
nums = [7,5,6,8,3]

# Example Output 1
1

# Example Input 2
nums = [3,3]

# Example OUtput 2
3
```
